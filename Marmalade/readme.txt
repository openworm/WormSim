C. elegans simulator app using Marmalade.
User tunes the worm's nervous system to allow it to find food.
More info at: http://wormworx.portegys.com
Instructions:
1. Download and install Marmalade: https://www.madewithmarmalade.com/
2. Open the WormWorx.mkb project file from the Marmalade Hub.
3. Run in simulator.
